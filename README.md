
## Requirements

* Kotlin-enabled project
* Jetpack Compose-enabled project
* An [API key][api-key]
* API level 24+

## App

To build the app:

1. Get a [API key][api-key]
2. Create a file in the root directory named `local.properties` , replacing YOUR_KEY with the key from step 1: `API_KEY=YOUR_KEY`
3. Build and run
